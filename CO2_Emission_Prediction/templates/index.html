About This Project

This machine learning application uses historical data from global indicators to predict CO2 emissions by country. Our model incorporates factors such as GDP, population, energy usage, and industrial activity to generate accurate forecasts.

The prediction system is powered by a Random Forest Regressor model trained on World Bank data indicators from 2000-2019. The model achieved 99.85% accuracy (R¬≤ score) on test data, making it a reliable tool for emissions forecasting.
